This is a QA harness. Every one of the sixteen enforceable interactive components appears below,
each pushed with realistic props and then deliberately stressed with long text, many items, and
edge values. The prose here is intentionally verbose so that inline components like Annotations and
Sidenotes have real running text to attach to, and so that wrapping and reflow behavior can be
judged against a genuine paragraph rather than a single short line that hides bugs.
